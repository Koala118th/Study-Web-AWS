# Study Web AWS

This project is a web application built with **React + TypeScript + Vite** to document and share knowledge about AWS services (S3, EC2, etc.) among team members.

---

# Tech Stack

- React
- TypeScript
- Vite
- Docker
- Docker Compose

---

# Getting Started

You can run this project in **two ways**:

---

## Option 1: Run with Docker (Recommended for team)

### 🔹 Prerequisites
- Install Docker Desktop

---

### 🔹 Steps

1. Clone the repository:

```bash
git clone https://github.com/Koala118th/Study-Web-AWS.git
cd Study-Web-AWS
```

2. Run the app using Docker Compose:

```bash
docker-compose up
```

3. Open your browser:

```bash
http://localhost:5173
```
